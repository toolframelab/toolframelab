# ToolFrameLab Website

ToolFrameLab is a personal tool lab for small utilities, prompts, and experiments.

This repository contains the static website for ToolFrameLab, including the homepage and public tool pages.

## About

ToolFrameLab collects small but practical workflows and tools, then turns them into cleaner, easier-to-use versions.

The first public tool is **命盤整理小幫手**, an iOS Shortcut helper that formats chart data collected by the user into an AI-friendly prompt.

## Tool: 命盤整理小幫手

命盤整理小幫手 helps users format astrology / chart data into a structured prompt that can be pasted into AI tools.

It currently supports data collected from:

Astrodoor
Zydx
Human Design Report
文墨天機

ToolFrameLab is not affiliated with these third-party websites or apps.
The links and references are provided for transparency and user convenience.

## Privacy

命盤整理小幫手 runs as an iOS Shortcut.

Chart data is mainly processed on the user’s own device.
ToolFrameLab does not actively receive, store, or build a database of users’ chart data.

Users are encouraged to review, manage, and delete their own generated files after use.
